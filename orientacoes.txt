-----------Configuração para o projeto-----------
1- Crie um repositório vazio com o seu usuário ( não adicione arquivo readme nem o git ignore)
2- Clone o repositório https://github.com/techcomkathia/petLorem2.git para sua máquina
3- Faça as remoções e configurações do repositório petLorem2 que você acabou de clonar com os comandos abaixo:

git remote remove origin
git remote add origin https://github.com/owner/repo.git (coloque a url do repositório que você criou no seu usuário)
git config user.name "Seu Nome"
git config user.email "seu@exemplo.com"

4- Seguindo esses passos o repositório base da atividade agora estará configurado apontando para o repositório correto, do seu usuário. 

5- Observar o conteúdo do html e ler o que se pede:
5.1  Criar o nav bar com o componente https://getbootstrap.com/docs/5.0/components/navbar/#nav
fazendo a substituição dos hrefs pelos ids das sessões e do texto Navbar, pela imagem de logo. 

5.2 Criar carrossel com base no exemplo final: https://techcomkathia.github.io/PetLoremFinal/
https://getbootstrap.com/docs/5.0/components/carousel/#with-captions
Substituindo as imagens e textos necessários.
